# 哲学家百科全书 (Philosopher Encyclopedia)

![License](https://img.shields.io/badge/license-MIT-blue.svg) ![Status](https://img.shields.io/badge/status-持续更新-green.svg)

## 📖 项目简介

这是一个开源的哲学家百科项目，致力于系统性地介绍古今中外重要哲学家的：

- **生平经历** - 时代背景与个人轨迹
- **核心思想** - 哲学体系与关键观点
- **代表作品** - 主要著作及内容概要
- **历史影响** - 对后世哲学、社会、文化的深远影响

## 🎯 项目目标

- 为中文互联网提供高质量、结构化的哲学家介绍
- 帮助哲学爱好者系统学习哲学史
- 建立可协作、可扩展的开源知识库
- 每篇条目都经过认真考据和整理

## 📚 推荐阅读顺序（学习路径）

本项目不按时间排序，而是按**最适合学习哲学**的顺序排列。以下是一条经过设计的路径：

| 顺序 | 哲学家 | 为什么这样排 |
|------|--------|-------------|
| 1 | 阿尔贝·加缪 | 最易入门——"荒诞感"人人有体会，不需要哲学基础 |
| 2 | 让-保罗·萨特 | 承接加缪，系统展开存在主义的核心命题 |
| 3 | 索伦·克尔凯郭尔 | 回溯源头——"存在主义之父"，理解"个体"概念的起点 |
| 4 | 弗里德里希·尼采 | 另一条根——强力意志与上帝之死，与克尔凯郭尔并列为思想源头 |
| 5 | 马丁·海德格尔 | 理论奠基——《存在与时间》是萨特存在主义的理论底座，最硬核 |
| 6 | 西蒙娜·德·波伏娃 | 存在主义的应用——将"自由"框架推向女性主义 |
| 7 | 莫里斯·梅洛-庞蒂 | 现象学身体转向——补全法国存在主义圈的版图中 |
| ↓ 后续待添加 | 柏拉图、亚里士多德、康德、黑格尔…… | 古代与德国古典哲学，为深入学习打下基础 |
