# 哲学家百科全书 (Philosopher Encyclopedia)

![License](https://img.shields.io/badge/license-MIT-blue.svg) ![Status](https://img.shields.io/badge/status-持续更新-green.svg)

## 📖 项目简介

这是一个开源的哲学家百科项目，致力于系统性地介绍古今中外重要哲学家的：

- **生平经历** - 时代背景与个人轨迹
- **核心思想** - 哲学体系与关键观点
- **代表作品** - 主要著作及内容概要
- **历史影响** - 对后世哲学、社会、文化的深远影响

## 🎯 项目目标

- 为中文互联网提供高质量、结构化的哲学家介绍
- 帮助哲学爱好者系统学习哲学史
- 建立可协作、可扩展的开源知识库
- 每篇条目都经过认真考据和整理

## 📚 推荐阅读顺序（学习路径）

本项目不按时间排序，而是按**最适合学习哲学**的顺序排列。以下是一条经过设计的路径：

### 第一阶：存在主义入门（最容易上手）

| 顺序 | 哲学家 | 为什么这样排 |
|------|--------|-------------|
| 1 | [阿尔贝·加缪](philosophers/camus.md) | 最易入门——"荒诞感"人人有体会，不需要哲学基础 |
| 2 | [让-保罗·萨特](philosophers/sartre.md) | 承接加缪，系统展开存在主义的核心命题 |
| 3 | [索伦·克尔凯郭尔](philosophers/kierkegaard.md) | 回溯源头——"存在主义之父"，理解"个体"概念的起点 |
| 4 | [弗里德里希·尼采](philosophers/nietzsche.md) | 另一条根——强力意志与上帝之死，与克尔凯郭尔并列为思想源头 |
| 5 | [亚瑟·叔本华](philosophers/schopenhauer.md) | 尼采的思想源头——意志哲学与悲观主义 |
| 6 | [马丁·海德格尔](philosophers/heidegger.md) | 理论奠基——《存在与时间》是萨特存在主义的理论底座 |
| 7 | [西蒙娜·德·波伏娃](philosophers/beauvoir.md) | 存在主义的应用——将"自由"框架推向女性主义 |
| 8 | [莫里斯·梅洛-庞蒂](philosophers/merleau-ponty.md) | 现象学身体转向——补全法国存在主义圈的版图 |

### 第二阶：扩展与深化

| 顺序 | 哲学家 | 为什么这样排 |
|------|--------|-------------|
| 9 | [卡尔·雅斯贝尔斯](philosophers/jaspers.md) | 存在主义另一条路线——强调边界情境与交往理性 |
| 10 | [费奥多尔·陀思妥耶夫斯基](philosophers/dostoevsky.md) | 存在主义文学源头——《地下室手记》深刻影响尼采和萨特 |
| 11 | [格奥尔格·威廉·弗里德里希·黑格尔](philosophers/hegel.md) | 德国古典哲学集大成者，理解马克思和存在主义的必要背景 |
| 12 | [卡尔·马克思](philosophers/marx.md) | 历史唯物主义，萨特后期试图融合马克思主义 |
| 13 | [汉娜·阿伦特](philosophers/arendt.md) | 政治哲学——海德格尔的学生，对极权主义的深刻分析 |
| 14 | [米歇尔·福柯](philosophers/foucault.md) | 后结构主义——对存在主义的继承与批判，权力/知识分析 |
| 15 | [雅克·德里达](philosophers/derrida.md) | 解构主义——后现代思想的核心人物 |
| 16 | [路德维希·维特根斯坦](philosophers/wittgenstein.md) | 分析哲学转向——语言游戏，与大陆哲学形成对照 |
