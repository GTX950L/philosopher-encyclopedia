## 📖 项目简介

这是一个开源的哲学家百科项目，致力于系统性地介绍古今中外重要哲学家的：

- **生平经历** - 时代背景与个人轨迹
- **核心思想** - 哲学体系与关键观点
- **代表作品** - 主要著作及内容概要
- **历史影响** - 对后世哲学、社会、文化的深远影响

## 🎯 项目目标

- 为中文互联网提供高质量、结构化的哲学家介绍
- 帮助哲学爱好者系统学习哲学史
- 建立可协作、可扩展的开源知识库
- 每篇条目都经过认真考据和整理

## 📚 推荐阅读顺序（学习路径）
本项目不按时间排序，而是按**最适合学习哲学**的顺序排列。以下是一条经过设计的路径：
### 第一阶：存在主义入门（最容易上手）

| 顺序 | 哲学家 | 为什么这样排 |
|------|--------|-------------|
| 1 | [阿尔贝·加缪](philosophers/camus.md) | 最易入门——'荒诞感'人人有体会，不需要哲学基础 |
| 2 | [让-保罗·萨特](philosophers/sartre.md) | 承接加缪，系统展开存在主义的核心命题 |
| 3 | [索伦·克尔凯郭尔](philosophers/kierkegaard.md) | 回溯源头——'存在主义之父'，理解'个体'概念的起点 |
| 4 | [弗里德里希·尼采](philosophers/nietzsche.md) | 另一条根——强力意志与上帝之死，与克尔凯郭尔并列为思想源头 |
| 5 | [亚瑟·叔本华](philosophers/schopenhauer.md) | 尼采的思想源头——意志哲学与悲观主义 |
| 6 | [埃德蒙德·胡塞尔](philosophers/husserl.md) | 现象学创始人——先验还原与意向性，海德格尔的思想源头 |
| 7 | [马丁·海德格尔](philosophers/heidegger.md) | 理论奠基——《存在与时间》是萨特存在主义的理论底座 |
| 8 | [西蒙娜·德·波伏娃](philosophers/beauvoir.md) | 存在主义的应用——将'自由'框架推向女性主义 |
| 9 | [莫里斯·梅洛-庞蒂](philosophers/merleau-ponty.md) | 现象学身体转向——补全法国存在主义圈的版图 |

## 📁 项目结构

```
philosopher-encyclopedia/
├── README.md              # 项目说明
├── CONTRIBUTING.md        # 贡献指南
├── requirements.txt       # Python 依赖
├── CHANGELOG.md           # 更新日志
├── mkdocs.yml             # Zensical 站点配置（兼容 MkDocs 格式）
├── .editorconfig          # 编辑器格式配置
├── .github/
│   ├── workflows/         # CI/CD 工作流
│   ├── dependabot.yml     # 自动依赖更新
│   └── ISSUE_TEMPLATE/    # Issue 模板
├── philosophers/          # 哲学家条目目录（45 位）
│   ├── camus.md          # 阿尔贝·加缪
│   ├── sartre.md         # 让-保罗·萨特
│   └── ...               # 更多哲学家
├── docs/                  # 补充资料
│   ├── timeline.md        # 哲学史时间线
│   └── schools.md         # 哲学流派概览
├── stylesheets/
│   └── extra.css          # 自定义样式
├── scripts/
│   └── gen_index.py       # 一致性校验（只读，不改文件）
├── tags.md                # 标签分类页面
└── template.md            # 新条目模板
```

> **说明**：`mkdocs.yml` 被 [Zensical](https://zensical.org)（Material for MkDocs 原班团队打造的下一代 SSG）原生读取。
