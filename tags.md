# 🏷️ 标签分类

> 按哲学流派、时代、地域标签浏览所有哲学家。
>
> 标签由 MkDocs tags 插件自动生成，在每条哲学家条目的 front matter 中设置 `tags` 字段即可归类。

---

## 使用说明

在每个哲学家条目的 YAML front matter（文件最开头的 `---` 区块）中添加标签：

```yaml
---
tags:
  - 存在主义
  - 20世纪
  - 法国
---
```

然后在 mkdocs.yml 中启用 tags 插件，本页会自动聚合所有标签并生成索引。
