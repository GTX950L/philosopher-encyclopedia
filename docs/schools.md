## 📚 流派详解

### 1. 古希腊哲学 (Classical Greek Philosophy)

**时期**：公元前6世纪 - 公元前3世纪

**核心特征**：从自然哲学（追问宇宙本原）转向伦理政治哲学（追问"人应当如何生活"）

| 哲学家 | 核心贡献 |
|--------|---------|
| 苏格拉底 | 诘问法，德性即知识，"未经审视的人生不值得过" |
| 柏拉图 | 理念论，学园派，对后世影响最深远的哲学家 |
| 亚里士多德 | 逻辑学开创者，德性伦理学，百科全书式学者 |
| 赫拉克利特 | 万物流变（Panta Rhei）；逻各斯；对立统一；朴素辩证法的奠基者 |

**对后世影响**：整个西方哲学传统都可视为"柏拉图的注脚"（怀特海语）。

---

### 2. 中国先秦哲学 (Ancient Chinese Philosophy)

**时期**：公元前6世纪 - 公元前3世纪（轴心时代）

**核心特征**：关注现世伦理与政治秩序，较少形而上学思辨

| 哲学家 | 流派 | 核心概念 |
|--------|------|---------|
| 孔子 | 儒家 | 仁、礼、君子 |
| 老子 | 道家 | 道、无为、自然 |
| 庄子 | 道家 | 齐物、逍遥、坐忘 |
| 墨子 | 墨家 | 兼爱、非攻、尚贤、节用；三表法（中国逻辑学先声） |

**对后世影响**：儒家成为中国官方意识形态两千年；道家深刻影响中国文学、艺术和禅宗。

---

### 3. 理性主义 (Rationalism)

**时期**：17世纪

**核心主张**：知识根本上来源于理性，而非感官经验。"我思故我在"（笛卡尔）

| 哲学家 | 代表著作 |
|--------|---------|
| 笛卡尔 | 《第一哲学沉思录》 |
| 斯宾诺莎 | 《伦理学》；实体即神即自然（泛神论）；以几何方法推演伦理 |
| 莱布尼茨 | 《单子论》；充足理由律；预存和谐；「最佳可能世界」 |

**与经验主义之争**：理性主义 vs 经验主义，这场争论直到康德才得到"调和"。

---

### 4. 经验主义 (Empiricism)

**时期**：18世纪（英国）

**核心主张**：一切知识来源于感官经验，否认先天观念。

| 哲学家 | 核心问题 |
|--------|---------|
| 大卫·休谟 | 因果问题、归纳问题、自我问题 |

**对后世影响**：休谟直接启发康德"从独断论中醒来"；也影响20世纪分析哲学。

---

### 5. 德国古典哲学 (German Idealism)

**时期**：18世纪末 - 19世纪初

**核心特征**：以"先验"方法探讨理性、自由、历史等问题，体系宏大。

| 哲学家 | 核心体系 |
|--------|---------|
| 伊曼努尔·康德 | 三大批判（纯粹理性、实践理性、判断力） |
| 格奥尔格·威廉·弗里德里希·黑格尔 | 辩证法、绝对精神、历史哲学 |

**对后世影响**：黑格尔深刻影响马克思；康德被誉为"哲学界的哥白尼"。

---

### 6. 存在主义 (Existentialism)

**时期**：19世纪（先驱）→ 20世纪（鼎盛）

**核心命题**：存在先于本质；人是被判定为自由的；荒诞；焦虑；本真性（Authenticity）。

#### 先驱（19世纪）

| 哲学家 | 核心洞见 |
|--------|---------|
| 索伦·克尔凯郭尔 | "单一者"；信仰的飞跃；绝望是精神上的死亡 |
| 弗里德里希·尼采 | 上帝已死；超人；权力意志；永恒轮回 |
| 亚瑟·叔本华 | 意志是世界的本质；悲观主义；审美救赎 |
| 费奥多尔·陀思妥耶夫斯基 | 文学化的存在主义；地下室人；自由与恶 |

#### 鼎盛（20世纪）

| 哲学家 | 代表著作 |
|--------|---------|
| 马丁·海德格尔 | 《存在与时间》——存在主义的理论奠基 |
| 卡尔·雅斯贝尔斯 | 边界情境；交往理性 |
| 让-保罗·萨特 | 《存在与虚无》——"人被判自由" |
| 阿尔贝·加缪 | 《西西弗神话》——荒诞哲学 |
| 西蒙娜·德·波伏娃 | 《第二性》——将存在主义推向女性主义 |
| 莫里斯·梅洛-庞蒂 | 《知觉现象学》——身体现象学 |

---

### 7. 现象学 (Phenomenology)

**时期**：20世纪初兴起

**核心方法**："回到事物本身"——悬搁自然态度，描述意识如何构成意义。

| 哲学家 | 核心贡献 |
|--------|---------|
| 埃德蒙德·胡塞尔 | 现象学创始人；意向性理论 |
| 马丁·海德格尔 | 将现象学从认识论转向存在论（《存在与时间》） |

**对后世影响**：现象学深刻影响存在主义、解释学、认知科学。

---

### 8. 后结构主义 / 后现代主义 (Post-structuralism / Postmodernism)

**时期**：20世纪60年代起（法国）

**核心主张**：批判"宏大叙事"；解构二元对立；权力/知识分析；差异先于同一。

| 哲学家 | 核心概念 |
|--------|---------|
| 米歇尔·福柯 | 权力/知识；规训；主体化；性史 |
| 雅克·德里达 | 解构；延异；药（pharmakon） |

**与存在主义的关系**：既继承（对主体性的关注）又批判（反对"人本主义"）。

---

### 9. 分析哲学 (Analytic Philosophy)

**时期**：20世纪初起（英美）

**核心特征**：语言转向；逻辑分析；清晰性优先；反对"形而上学废话"。

| 哲学家 | 核心贡献 |
|--------|---------|
| 伯特兰·罗素 | 数理逻辑；逻辑原子主义；描述语理论 |
| 路德维希·维特根斯坦 | 《逻辑哲学论》（早期）；《哲学研究》（后期，语言游戏） |
| 戈特洛布·弗雷格 | 现代逻辑之父；《概念文字》；涵义与指称；逻辑主义纲领 |

**与大陆哲学的对立**：分析 vs 大陆，这条"两种哲学文化"的分裂至今仍在。

---

### 10. 政治哲学 (Political Philosophy)

**跨流派**，本项目的政治哲学家跨越多个时代：

| 哲学家 | 核心理论 |
|--------|---------|
| 托马斯·霍布斯 | 《利维坦》；自然状态是"所有人对所有人的战争" |
| 约翰·洛克 | 自然权利；有限政府；反抗权 |
| 约翰·斯图尔特·密尔 | 《论自由》；伤害原则；功利主义 |
| 汉娜·阿伦特 | 《极权主义的起源》；行动理论；公共领域 |
| 约翰·罗尔斯 | 《正义论》；原初状态与无知之幕；两个正义原则（差异原则） |

---

### 11. 实用主义 (Pragmatism)

**时期**：20世纪（美国）

**核心主张**：真理是"有用的信念"；意义在于实际后果；民主作为一种生活方式。

| 哲学家 | 核心著作 |
|--------|---------|
| 约翰·杜威 | 《民主与教育》；工具主义认识论 |

---

### 12. 批判理性主义 (Critical Rationalism)

**时期**：20世纪

**核心主张**：科学理论不能被证实，只能被证伪；"开放社会"对抗"封闭社会"。

| 哲学家 | 核心著作 |
|--------|---------|
| 卡尔·波普尔 | 《科学发现的逻辑》；《开放社会及其敌人》 |

---

## 🔗 流派关系图

| 关系 | 说明 |
|------|------|
| 胡塞尔 → 海德格尔 | 现象学 → 存在主义现象学 |
| 克尔凯郭尔/尼采 → 萨特/加缪 | 存在主义先驱 → 存在主义鼎盛 |
| 黑格尔 → 马克思 | 辩证法 → 历史唯物主义 |
| 笛卡尔/休谟 → 康德 | 理性/经验 → 批判哲学综合 |
| 萨特 → 福柯/德里达 | 存在主义 → 后结构主义（继承与批判） |
| 罗素 → 维特根斯坦 | 分析哲学师生传承（早期）；后期维特根斯坦批判罗素 |
| 海德格尔 → 阿伦特 | 师生；政治哲学与存在主义的结合 |
| 老子/庄子 | 道家内部传承；"老庄"并称 |

## 📖 推荐阅读路径（按流派）

### 想了解存在主义？
→ 加缪（《西西弗神话》）→ 萨特（《存在与虚无》入门读《存在主义是一种人道主义》）→ 海德格尔（《存在与时间》选读）→ 克尔凯郭尔（《恐惧与颤栗》）

### 想了解分析哲学？
→ 罗素（《西方哲学史》通俗易懂）→ 维特根斯坦（《逻辑哲学论》→《哲学研究》）

### 想了解中国哲学？
→ 孔子（《论语》）→ 老子（《道德经》）→ 庄子（《庄子》内篇）→ 墨子（《墨子·兼爱》）

### 想了解政治哲学？
→ 霍布斯（《利维坦》第一章）→ 洛克（《政府论》下篇）→ 密尔（《论自由》）→ 阿伦特（《人的境况》）→ 罗尔斯（《正义论》）
